# AI Film OS v3

מערכת מודולרית לניהול הפקת סרטי AI.

## כלול בגרסה זו

- Dashboard
- Story Bible / Assets
- Scenes
- Shots
- Prompt Builder
- Continuity Checks
- SQLite
- API מודולרי
- ממשק RTL

## הפעלה

```bash
pip install -r requirements.txt
python run.py
```

פתחו:

```text
http://localhost:8000
```
