# AI Film OS — „כתובת אפס”

סוכן OpenAI Agents SDK ייעודי להפקת הטריילר והסרט „כתובת אפס”.

מערכת הפקה בעברית המשלבת את סוכן ההפקה עם ניהול סצנות ושוטים, ספריית נכסים, היסטוריית פרומפטים ותוצאות מדיה, ובדיקות רציפות מובנות.

## יכולות Film OS v3

- סצנות המקושרות לשוטים, כולל סדר, סטטוס, תקציר ומשך מצטבר.
- שדות שוט מובנים: משך, גודל שוט, זווית, עדשה, תנועה, קומפוזיציה, פעולה, תאורה, צבע ואודיו.
- גרסה חדשה נשמרת אוטומטית בכל שינוי בפרומפט או ביצירת דף הפקה בעזרת הסוכן.
- תוצאות תמונה ווידאו נשמרות כרשומות ממוספרות בלי לדרוס תוצאות קודמות.
- Continuity QA לפי קטגוריה, חומרה, סטטוס, מצב צפוי, מצב שנמצא ופתרון.
- ממשק RTL רספונסיבי בעברית.

השדרוג משתמש ב־SQLite migrations מצטברים בלבד. מסד נתונים קיים נשמר, והשדות והטבלאות החדשים מתווספים בהפעלה הראשונה.

## הפעלה בדפדפן

```bash
PORT=8000 uv run python3 main.py
```

פתחו `http://localhost:8000`.

## הפעלה בשורת פקודה

```bash
uv run python3 main.py
```

דוגמאות לבקשות:

- `הכיני דף הפקה מלא לשוט 1.`
- `כתבי פרומפט Magnific מדויק לשוט 6.`
- `בדקי האם ההצעה הזאת לשוט 15 סותרת את חוקי העולם.`
- `הציגי את סטטוס כל 20 השוטים.`

המפתח נשמר מחוץ לחבילת הפרויקט ואינו כלול בקוד.

## בדיקות

```bash
uv run python -m unittest -v
```
